# BookRegister version 1.0.0

Is the final assigment of Unit Test class for the course React22K of Helsinki Bussisnes Collage, made by Victor Griñán.

## Steps

- clone the repo, cd into it.

- type in your console:

```shell
    npm i
```

- run the program:

```shell
   node index.js
```

- run the test:

```shell
   npm test
```
